# King's Cross Viber

A fast hackathon web app for planning King’s Cross nights out from a small local graph.

The demo mirrors the sponsor story from the guide:

- Kimchi: the low-cost reasoning layer you can narrate during the demo.
- Neo4j: represented by the place graph in `data/places.json`.
- Tessl: represented by reusable agent instructions in `tessl.json` and the local setup.

## Run

```bash
npm run dev
```

Open `http://127.0.0.1:5173`.

## Test

```bash
npm test
```
